Searching Types

Searching algorithms are used to find a specific value within a data structure.

  • Linear Search: Sequentially checks each element until the desired element is found.
  • Binary Search: Efficiently finds an element in a sorted array by repeatedly dividing the search interval in half.

Linear and Binary Search

Linear Search

Linear search goes through each element one by one.

function linearSearch(arr, target) {
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === target) {
            return i; // Return the index of the target
        }
    }
    return -1; // Target not found
}

Binary Search

Binary search works only on sorted arrays.

function binarySearch(arr, target) {
    let left = 0;
    let right = arr.length - 1;

    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        if (arr[mid] === target) {
            return mid; // Target found
        } else if (arr[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return -1; // Target not found
}

Sorting Algorithms

Sorting algorithms arrange the elements of a list in a specific order.

  • Selection Sort: Repeatedly selects the minimum element and moves it to the sorted portion.
  • Bubble Sort: Repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.
  • Insertion Sort: Builds the final sorted array one item at a time.
  • Heap Sort: Converts the array into a heap structure and then sorts it.

Example of Selection Sort:

function selectionSort(arr) {
    for (let i = 0; i < arr.length - 1; i++) {
        let minIndex = i;
        for (let j = i + 1; j < arr.length; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        // Swap
        [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
    }
    return arr;
}

Hash Functions

Hash functions map data of arbitrary size to fixed-size values.

  • Types of Hash Functions:
    • Division Method
    • Multiplication Method
    • Universal Hashing
  • Collision: Occurs when two different inputs produce the same hash value.
  • Collision Resolution Techniques:
    • Chaining: Stores multiple values in a single hash table index.
    • Open Addressing: Finds another open slot in the hash table.
  • Perfect Hashing: A hashing scheme that avoids collisions entirely.
