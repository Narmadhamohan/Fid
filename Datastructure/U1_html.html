Introduction to Web Pages

Web pages are the fundamental building blocks of the internet. They are documents that can be displayed in a web browser and are written using HTML (HyperText Markup Language).

Basics of HTML

Elements and Tags

  • HTML documents are made up of elements defined by tags.
  • Common tags include:
    • <html> - The root element
    • <head> - Contains meta-information
    • <body> - Contains the content of the document

HTML5 Features

  • New semantic elements: <header>, <footer>, <article>, <section>
  • Improved support for multimedia: Audio and Video elements
  • Canvas for graphics and animation

Business Case Scenario

Imagine a local bakery wants to create a simple website to showcase their products. They will use HTML to create pages with headings, paragraphs, and images to attract customers.

HTML Tables

Creating Tables

  • Tables are created using the <table> tag.
  • Use <tr> for table rows, <td> for table data (cells).

Table Attributes

  • Cellpadding: Space between cell content and cell border
  • Cellspacing: Space between cells
  • Border: Width of the table border
  • Rowspan: Merges cells vertically
  • Colspan: Merges cells horizontally

Images in HTML

Inserting Images

  • Use the <img> tag with src, height, and width attributes.

Anchor Elements

  • The <a> tag is used to create hyperlinks.

Audio and Video Elements

  • Use the <audio> tag to embed sound.

Basics of CSS

What is CSS?

Cascading Style Sheets (CSS) is used to style HTML documents.

Types of CSS

  • Inline CSS: Directly within HTML tags.
  • Internal CSS: Inside <style> tags in the <head> section.
  • External CSS: In a separate .css file linked to HTML.

CSS Selectors

  • Element Selector: Selects elements by their tag name.
  • Class Selector: Selects elements with a specific class.
  • ID Selector: Selects a unique element with a specific ID.

Common CSS Properties

  • Text: color, font-size, text-align
  • Border: border-style, border-width
  • Table: border-collapse, border-spacing
  • Image: width, height, object-fit
  • Padding: padding-top, padding-bottom
