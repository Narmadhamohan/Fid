Introduction to Circular Queue

  • Definition: A circular queue is a linear data structure that connects the last position back to the first position.
  • Advantage: Efficiently utilizes space by allowing reuse of empty slots.
  • Key Operations:
    • Enqueue: Add an element to the rear.
    • Dequeue: Remove an element from the front.

Calculating Circular Queue Formula

Formula Derivation:

To keep track of the front and rear positions in a circular queue, we use modular arithmetic.

Key Variables:

  • Front: Points to the front element.
  • Rear: Points to the last element.
  • Size: Total number of elements currently in the queue.
  • Capacity: Maximum number of elements the queue can hold.

Enqueue Operation:

When adding an element, the formula used is:

rear = (rear + 1) % capacity

Dequeue Operation:

When removing an element, the formula used is:

front = (front + 1) % capacity

Time Complexity:

  • Enqueue: O(1)
  • Dequeue: O(1)

Introduction to Linked List

  • Definition: A linked list is a linear data structure consisting of a sequence of elements where each element points to the next.
  • Components: Nodes, which contain data and a reference to the next node.
  • Types:
    • Singly Linked List
    • Doubly Linked List
    • Circular Linked List

Operations on Linked Lists

  • Insertion: Adding a new node.
  • Deletion: Removing a node.
  • Traversal: Accessing each node to read data.
  • Search: Finding a node by its value.

Singly Linked List Operations

  • Insertion at Head: O(1)
  • Insertion at Tail: O(n)
  • Deletion: O(n)
  • Traversal: O(n)

Doubly Linked Lists & Circular Lists

  • Doubly Linked List: Each node points to both the next and previous nodes.
  • Circular Linked List: Last node points back to the first node.
  • Advantages: Efficient bidirectional traversal and easier deletion.

Operations of Linked List using Stacks

  • Push: Add a node to the top of the stack.
  • Pop: Remove the top node and return its value.
  • Peek: View the top node without removing it.

Operations of Linked List using Queues

  • Enqueue: Add a node to the end of the queue.
  • Dequeue: Remove the node from the front of the queue.
  • Peek: View the front node without removing it.
