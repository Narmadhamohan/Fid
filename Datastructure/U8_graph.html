Introduction to Graphs

A graph is a collection of nodes (vertices) connected by edges. There are two main types of graphs:

  • Directed Graph: Edges have a direction.
  • Undirected Graph: Edges have no direction.

Example: A simple graph with vertices A, B, C, and edges AB, AC.

Graph Traversals & Kruskal's Algorithm

Graph traversals are methods to visit all vertices in a graph:

  • Breadth-First Search (BFS): Visits all neighbors before moving to the next level.
  • Depth-First Search (DFS): Visits as far as possible down one branch before backtracking.

Kruskal's Algorithm: Used to find the Minimum Spanning Tree (MST) of a graph.

Rules:

  • Sort all edges in non-decreasing order of their weights.
  • Pick the smallest edge. Check if it forms a cycle with the spanning tree formed so far.
  • If it doesn't form a cycle, include this edge in the MST.
  • Repeat until there are (V - 1) edges in the MST (where V is the number of vertices).

Example: Given edges with weights, find the MST.

Input: Edges (A-B, 1), (A-C, 3), (B-C, 2)

Output: Minimum Spanning Tree includes edges A-B, B-C.

Shortest Path Algorithm

Dijkstra's Algorithm: Finds the shortest path from a starting vertex to all other vertices.

Rules:

  • Initialize distances from the source to all vertices as infinity, except the source itself (which is 0).
  • Use a priority queue to repeatedly select the vertex with the smallest distance.
  • Update the distances to neighboring vertices if a shorter path is found.

Example: Given a weighted graph, calculate the shortest path.

Input: Graph with vertices A, B, C and edges A-B (1), A-C (4), B-C (2)

Output: Shortest path from A to C is A-B-C with a distance of 3.

Implementation of Graphs

Graph Representation:

  • Adjacency Matrix: A 2D array where each cell (i, j) indicates if there is an edge from vertex i to vertex j.
  • Adjacency List: A list of lists where each list corresponds to a vertex and contains a list of its adjacent vertices.

Example: Represent a graph using an adjacency list.

Input: Vertices A, B, C with edges A-B, A-C.

Output: Adjacency List: A -> [B, C], B -> [A], C -> [A].
