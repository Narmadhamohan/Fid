Full Binary Tree

  • Definition: A full binary tree is a binary tree in which every node other than the leaves has exactly two children.
  • Characteristics:
    • All leaf nodes are at the same level.
    • The number of leaf nodes is always one more than the number of internal nodes.

Formula: If 'n' is the number of internal nodes, the total number of nodes (N) can be calculated as:

N = 2n + 1

Perfect Binary Tree

  • Definition: A perfect binary tree is a binary tree in which all interior nodes have two children and all leaves are at the same level.
  • Characteristics:
    • Height of the tree is minimal.
    • All nodes at the last level are filled.

Formula: The number of nodes (N) in a perfect binary tree of height 'h' can be calculated as:

N = 2h+1 - 1

  • Level 1: 2 nodes
  • Level 2: 4 nodes
  • Level h: 2h nodes

Thus, the total number of nodes is:

N = 1 + 2 + 4 + ... + 2h = 2h+1 - 1

Balanced Binary Tree

  • Definition: A balanced binary tree is a binary tree in which the depth of the two subtrees of every node never differ by more than one.
  • Characteristics:
    • Height is minimized.
    • Better performance for operations like insertion, deletion, and searching.

Formula: The maximum height (h) of a balanced binary tree with 'n' nodes can be calculated as:

h ≤ log2(n + 1)
