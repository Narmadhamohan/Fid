PostgreSQL Index Pointer Types

Index Type Pointer Type Best For Data Types Query Example
B-tree (Default) Row-based (TIDs) General-purpose types (e.g., integers, strings, dates) SELECT * FROM employees WHERE salary > 50000;
SELECT * FROM employees WHERE name like '%jo'; (range query)
Hash Hash buckets Equality searches (e.g., integers, strings) SELECT * FROM users WHERE id = 123; (equality query)
GIN List of TIDs Full-text, arrays, JSONB SELECT * FROM documents WHERE content @@ to_tsquery('cat & dog'); (full-text query)
GiST Bounding boxes or ranges Geometric, range, nearest-neighbor SELECT * FROM locations ORDER BY geom <-> point '(1,1)' LIMIT 1; (nearest neighbor query)
BRIN Block ranges Large, ordered datasets (e.g., time-series) SELECT * FROM sensor_data WHERE timestamp > '2024-01-01'; (time-series query)
SP-GiST Partitions of data space Spatial partitioning, tree-like structures SELECT * FROM networks WHERE ip_address <@ '192.168.1.0/24'; (prefix query)
