Intro to web pages

1. Intro to Web Pages

A web page is a digital document that is written in HTML (HyperText Markup Language) and viewed in a web browser. It is a part of the World Wide Web, consisting of text, images, videos, and other multimedia elements that can be linked together using hyperlinks.

Static vs Dynamic Web Pages:

Static Web Pages: Written in HTML and display the same content to every visitor. Dynamic Web Pages: Can change content based on user interaction, often powered by server-side programming languages like PHP, Python, or JavaScript. Importance: Web pages are the building blocks of websites, serving information to users on the internet.

Elements, Tags

HTML Elements are the building blocks of HTML documents. They define the structure and content of a web page.
Tags: HTML uses tags to mark elements. These tags are enclosed in angle brackets.
Common HTML Elements:
<html> : Defines the root of the HTML document.
< head >: Contains meta-information, including the title, links to stylesheets, and scripts.
< body >: Contains the visible content of the web page like text, images, etc.
< p > : Paragraph tag.
<a >: Anchor tag for hyperlinks.
<img>: Tag for embedding images.

Structure of HTML Document

<!DOCTYPE html>: This is a declaration that defines the document as HTML5.
<html> : Root element that wraps all the content in the document.
< head >: Contains meta-information about the document such as the title, character encoding, and linked stylesheets or scripts.
< body >: Holds the visible content, including text, images, and multimedia.
