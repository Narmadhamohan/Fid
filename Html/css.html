Basics & Types of CSS

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML or XML. It controls the layout of multiple web pages all at once and separates the content from the presentation.

Types of CSS:

  • Inline CSS: Styles applied directly within the HTML element. 
    <p style="color: red;">This text is red.</p>
  • Internal CSS: Styles defined within a <style> tag in the <head> section. 
    <style>
p { color: blue; }
</style>
  • External CSS: Styles defined in a separate .css file linked to the HTML document. 
    <link rel="stylesheet" href="styles.css">

Different Ways of Selectors

Selectors in CSS are patterns used to select the elements you want to style. Here are some common types of selectors:

  • Element Selector: Selects elements based on the HTML tag name. 
    div { color: green; }
  • Class Selector: Selects elements with a specific class attribute. 
    .highlight { background-color: yellow; }
  • ID Selector: Selects a single element with a specific ID attribute. 
    #main-title { font-size: 24px; }
  • Attribute Selector: Selects elements based on a specific attribute. 
    [type="text"] { border: 1px solid #ccc; }
  • Pseudo-class Selector: Selects elements in a specific state. 
    a:hover { color: red; }
  • Pseudo-element Selector: Selects a specific part of an element. 
    p::first-line { font-weight: bold; }

Basic & Common Properties (Text & Border)

Here are some basic properties:

Text Properties:

  • color: Sets the color of the text. 
    p { color: blue; }
  • font-size: Sets the size of the font. 
    h1 { font-size: 36px; }
  • font-family: Specifies the font of the text. 
    body { font-family: Arial, sans-serif; }
  • text-align: Sets the horizontal alignment of the text. 
    h2 { text-align: center; }

Border Properties:

  • border: Shorthand property to set width, style, and color. 
    div { border: 1px solid black; }
  • border-radius: Rounds the corners of an element. 
    img { border-radius: 10px; }

Table & Image Properties

Common properties for tables and images:

Table Properties:

  • border-collapse: Specifies whether table borders should be collapsed into a single border. 
    table { border-collapse: collapse; }
  • padding: Sets the space between the cell content and the cell border. 
    td { padding: 10px; }

Image Properties:

  • width: Sets the width of the image. 
    img { width: 100%; }
  • height: Sets the height of the image. 
    img { height: auto; }

Padding & CSS Layout

Padding is the space between the content and the border of an element:

div {
  padding: 20px;
}

CSS Layout Techniques:

  • Flexbox: A layout model that allows you to design a layout structure without using float or positioning. 
    .container {
  display: flex;
  justify-content: space-between;
}
  • Grid: A layout model that provides a two-dimensional grid-based layout system. 
    .grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
  • Positioning: Allows you to control the layout through static, relative, absolute, fixed, or sticky positioning. 
    div {
  position: relative;
  top: 10px;
}
