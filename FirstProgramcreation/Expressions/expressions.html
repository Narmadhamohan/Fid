Mind Map: Expressions, Statements, and Blocks

Expressions
An expression is a combination of variables, operators, and method invocations that evaluates to a single value.

The data type of the value returned by an expression depends on the elements used in the expression.

Example:
int result = 1 + 2;
int result = 1 + 2; // result is now 3
if (value1 == value2)
System.out.println("value1 == value2");



an expression can return int and other types of values as well, such as boolean or String.

System.out.println("Hello World!");
Compound Expressions
  • Compound expressions are made up of smaller expressions.

  • example of a compound expression: 1 * 2 * 3

  • Order in which the expression is evaluated is: Important or not.??

    important if we have other type of expressions:

    x + y / 100 // ambiguous


  • Example(Unambiguous, Recommended):
    (x + y) / 100;


Code Tip! When writing compound expressions, be explicit and indicate with parentheses which operators should be evaluated first.

This practice makes code easier to read and to maintain.
Statements
Statements represent a complete unit of execution. They can be assignment, increment, or method invocation statements.

Example:
aValue = 8933.234;
aValue++;

Statements are roughly equivalent to sentences in natural languages.
