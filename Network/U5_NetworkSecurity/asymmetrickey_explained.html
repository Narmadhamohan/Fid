Asymmetric Encryption Example

Scenario: Alice and Bob Want to Send the Word "CAT"

Alice and Bob will use asymmetric encryption to securely send the word "CAT" using a simple number-based system.

Step 1: Assign Numbers to Letters

  • C = 3
  • A = 1
  • T = 20

So, the message "CAT" becomes the numbers: 3, 1, 20.

Step 2: Generate Keys

Alice’s Public Key: 7
Alice’s Private Key: 3
Bob’s Public Key: 11
Bob’s Private Key: 5

Step 3: Encrypt the Message ("CAT")

Bob wants to send "CAT" to Alice.

For "C" (3):
Encrypted letter = 3 × 7 = 21
For "A" (1):
Encrypted letter = 1 × 7 = 7
For "T" (20):
Encrypted letter = 20 × 7 = 140

So, the encrypted message Bob sends to Alice is: 21, 7, 140.

Step 4: Decrypt the Message

Alice receives the encrypted message and uses her private key (3) to decrypt it.

For "21":
Decrypted letter = 21 ÷ 3 = 7 (C)
For "7":
Decrypted letter = 7 ÷ 3 = 1 (A)
For "140":
Decrypted letter = 140 ÷ 3 = 20 (T)

Alice successfully decrypts the message "CAT".

Key Takeaways

  • Bob encrypts the message using Alice's public key.
  • Alice decrypts the message using her private key.
  • Only Alice can read the message sent by Bob.
