Types of Networks

Local Area Network (LAN)

  • A Local Area Network (LAN) connects computers within a limited area, such as a home, school, or office.
  • Typically spans a few hundred meters.
  • High data transfer rates, generally ranging from 100 Mbps to 10 Gbps.
  • Commonly used technologies: Ethernet, Wi-Fi.

Metro Area Network (MAN)

  • A Metro Area Network (MAN) connects multiple LANs within a specific geographic area, such as a city.
  • Typically spans up to 50 kilometers.
  • Used to connect various LANs in a metropolitan area.
  • Common applications include connecting government offices and educational institutions.

Wide Area Network (WAN)

  • A Wide Area Network (WAN) covers a large geographic area, connecting multiple LANs and MANs.
  • Can span cities, countries, or even continents.
  • Typical data transfer rates are much lower compared to LANs, often ranging from 56 Kbps to 1 Gbps.
  • Examples include the Internet and private corporate networks.

Personal Area Network (PAN)

  • A Personal Area Network (PAN) is a small network, typically within a range of a few meters.
  • Used for connecting devices like smartphones, tablets, and laptops.
  • Common technologies include Bluetooth and infrared.

Hypertext Transfer Protocol (HTTP)

  • HTTP is an application-layer protocol used for transmitting hypermedia documents on the web.
  • It facilitates communication between clients and servers.
  • HTTP works by establishing a connection between a web browser (client) and a web server.

Client-Server Architecture

  • The client-server model is a distributed application structure that separates tasks between service providers (servers) and service requesters (clients).
  • Clients request resources or services from the server, which processes the requests and sends back the appropriate response.
  • This architecture is fundamental to web applications, where web browsers act as clients and web servers provide data.

Communication

  • Communication in networking refers to the transfer of data between devices over a network.
  • It involves protocols that define how data is transmitted and received.
  • Effective communication ensures reliable and efficient data exchange.
