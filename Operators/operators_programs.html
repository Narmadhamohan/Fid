Slide 1: Introduction

Example Program 1:

int a = 10, b = 20, c = 30;

// Correct:
if (a > b && b < c || a < c) {
    System.out.println("Condition is true");
}

// Explanation:
- `&&` has higher precedence than `||`, so `a > b && b < c` is evaluated first.
- Then, the result of this expression is combined with `a < c` using the `||` operator.

Slide 2: Data Structures

Example Program 2:

int a = 5, b = 10, result;

// Correct:
result = (a > b) ? a : b;

// Explanation:
- The conditional operator `?:` is evaluated first because it has higher precedence than `=`.
- After deciding between `a` and `b`, the result is assigned to the `result` variable.

Slide 2:Example 3: Assignment (=) Associativity

Example Program 3:

int x, y, z;

// Correct:
x = y = z = 50; // Right-associative

// Explanation:
- Assignment operators are right-associative, meaning `z = 50` is evaluated first, then `y = z`, and finally `x = y`.



Example 4: Logical NOT (!) vs Logical AND (&&)

boolean a = true, b = false;

// Correct:
if (!a && b) {
    System.out.println("Condition is true");
}

// Explanation:
- The `!` (logical NOT) operator has higher precedence than `&&`, so `!a` is evaluated first.
- Then, the result of `!a` is combined with `b` using the `&&` operator.
