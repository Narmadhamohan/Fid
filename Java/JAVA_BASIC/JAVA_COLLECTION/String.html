Understanding String Comparison in Java

1. Using equals()

The equals() method checks for value equality. It compares the actual contents of the strings. 

      String str1 = "hello";
      String str2 = new String("hello");

      System.out.println(str1.equals(str2)); // Output: true

In this example, str1.equals(str2) returns true because the contents are the same.

2. Using == Operator

The == operator checks for reference equality. It determines whether both references point to the same object in memory. 

      String str1 = "hello";
      String str2 = new String("hello");

      System.out.println(str1 == str2); // Output: false

In this example, str1 == str2 returns false because they are different objects in memory.

3. Using compareTo()

The compareTo() method compares two strings lexicographically. It returns:

  • A negative integer if the first string is lexicographically less than the second.
  • Zero if the strings are equal.
  • A positive integer if the first string is lexicographically greater than the second.

      String str1 = "apple";
      String str2 = "banana";

      System.out.println(str1.compareTo(str2)); // Output: -1

In this example, str1.compareTo(str2) returns a negative value because "apple" is less than "banana".

What is Lexicographical Order?

Lexicographical order is similar to dictionary order. It compares the strings based on the Unicode value of each character. The comparison is done character by character from left to right.

For example, consider the following strings:

      "apple"  // Unicode values: 97, 112, 112, 108, 101
      "banana" // Unicode values: 98, 97, 110, 97, 110

Comparing "apple" and "banana":

  • The first character 'a' (Unicode 97) is less than 'b' (Unicode 98).
  • Since 'a' is less than 'b', "apple" is lexicographically less than "banana".

Summary of Results

      str1.equals(str2)   // true
      str1 == str2        // false
      str1.compareTo(str2) // -1 (if str1 is "apple" and str2 is "banana")

Conclusion

In Java, use equals() for value comparison, == for reference comparison, and compareTo() for lexicographic ordering. Lexicographical order compares strings based on the Unicode values of their characters.
