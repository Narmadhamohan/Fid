Introduction to Functions

What is a Function?

A function is a block of code designed to perform a particular task. Functions help in code reusability and modular programming.

  • Functions can take inputs (called parameters) and can return outputs (results).
  • Functions allow breaking down complex problems into smaller, manageable parts.
  • Java provides predefined functions like System.out.println(), or users can define their own functions.
Predefined and User Defined Functions

Predefined Functions

Predefined functions are built into Java, such as Math.sqrt(), System.out.println().

User Defined Functions

Functions created by the programmer are called user-defined functions. They are declared using the void or return type, followed by the function name and parameters.

Example: 

public class Calculator {
    // Predefined function: Math.sqrt()
    public void calculateSquareRoot(int number) {
        double result = Math.sqrt(number);
        System.out.println("Square Root: " + result);
    }
    
    // User-defined function
    public int add(int a, int b) {
        return a + b;
    }
}
Function Parameters and Function Call

Function Parameters and Function Call

  • Parameters: Functions can accept inputs called parameters. For example, int a and int b are parameters.
  • Function Call: Functions are called by their name, followed by parentheses. Example: add(10, 20);

public class FunctionExample {
    // Function with parameters
    public void greet(String name) {
        System.out.println("Hello, " + name);
    }

    public static void main(String[] args) {
        FunctionExample example = new FunctionExample();
        example.greet("John"); // Function call with argument
    }
}
