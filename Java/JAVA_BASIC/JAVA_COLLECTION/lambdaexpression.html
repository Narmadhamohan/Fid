Transforming Functions into Lambda Expressions

Step-by-Step Transformation

Step 1: Define a Regular Function

A regular function takes parameters and performs an action. Here’s an example: 

      // Regular function
      public int add(int a, int b) {
          return a + b;
      }

Step 2: Identify Parameters and Return Type

The function has two parameters (a and b) and returns an int.

Step 3: Convert to Lambda Expression

A lambda expression can be created using the same parameters and the body of the function. Here’s how the function looks as a lambda expression: 

      // Lambda expression
      (int a, int b) -> a + b

Step 4: Assign to a Functional Interface

The lambda expression can be assigned to a functional interface. For example: 

      // Functional interface
      @FunctionalInterface
      interface Adder {
          int add(int a, int b);
      }

      // Assigning lambda expression to functional interface
      Adder adder = (int a, int b) -> a + b;

Example Usage

You can now use the lambda expression like this: 

      int result = adder.add(5, 3);
      System.out.println(result); // Output: 8

Using Lambda Expressions with Comparable and Comparator

Example with Comparable

You can use lambda expressions to implement the Comparable interface for sorting: 

      class Person implements Comparable<Person> {
          String name;
          int age;

          Person(String name, int age) {
              this.name = name;
              this.age = age;
          }

          @Override
          public int compareTo(Person other) {
              return this.age - other.age; // Sorting by age
          }
      }

      // Usage
      List<Person> people = Arrays.asList(new Person("Alice", 30), new Person("Bob", 25));
      Collections.sort(people); // Sorts by age

Example with Comparator

You can also use lambda expressions to create a Comparator: 

      // Using Comparator with lambda expression
      Comparator<Person> compareByName = (p1, p2) -> p1.name.compareTo(p2.name);

      // Usage
      List<Person> people = Arrays.asList(new Person("Alice", 30), new Person("Bob", 25));
      Collections.sort(people, compareByName); // Sorts by name

Conclusion

Transforming functions into lambda expressions allows for more concise and readable code in Java. Lambda expressions provide a clear way to represent actions and behaviors in a functional style, including powerful sorting and comparison mechanisms through Comparable and Comparator.
