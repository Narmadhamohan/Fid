What are Generics?

Generics allow you to define classes, interfaces, and methods with a placeholder for types. This means you can create a data structure that can hold any type of object, making your code more flexible and type-safe.

For example, using List<T> enables you to create a list of any type: 

List<String> stringList = new ArrayList<>();

This ensures that only String objects can be added to stringList, providing compile-time type checking.
