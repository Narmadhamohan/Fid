Introduction to Arrays

What is an Array?

An array is a data structure used to store multiple values of the same type in a single variable.

  • Arrays can hold primitives and objects in Java.
  • Each item in an array is called an element, and each element is accessed by its index.
  • Arrays have fixed size in Java, meaning their length is defined when created and cannot be changed.
Example: 

public class ArrayExample {
    public static void main(String[] args) {
        // Declaration and initialization of an array
        int[] numbers = {1, 2, 3, 4, 5};

        // Access elements by index
        System.out.println("First element: " + numbers[0]);
        System.out.println("Third element: " + numbers[2]);
    }
}
Single Dimensional Arrays and Multi-Dimensional Arrays

Single Dimensional and Multi-Dimensional Arrays

  • Single Dimensional Array: It is a linear array where elements are stored in a single row.
  • Multi-Dimensional Array: It is an array of arrays, meaning each element is itself an array.

public class MultiDimensionalArrayExample {
    public static void main(String[] args) {
        // 2D array (multi-dimensional array)
        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        // Access elements in 2D array
        System.out.println("Element at (0,0): " + matrix[0][0]);
        System.out.println("Element at (2,2): " + matrix[2][2]);
    }
}
Matrices

What is a Matrix?

A matrix is a two-dimensional array, where each element is defined by a pair of indices (row and column).

Matrices are used in various mathematical and computational problems like graph representations, image processing, etc.
