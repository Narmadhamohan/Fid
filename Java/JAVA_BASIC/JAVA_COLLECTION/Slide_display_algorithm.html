Introduction to Algorithm

What is an Algorithm?

An algorithm is a step-by-step procedure to solve a problem or perform a task. In computer science, algorithms are a fundamental concept that forms the basis for programming and software development.

  • Characteristics of an Algorithm:
    • Finiteness: The algorithm must terminate after a finite number of steps.
    • Definiteness: Each step of the algorithm must be precisely defined.
    • Input: The algorithm receives input data.
    • Output: The algorithm produces output data.
    • Effectiveness: The steps should be basic enough to be carried out by a person or a machine.
Business Case Scenario - Finding the Shortest Route:

Imagine a company wants to optimize delivery routes for its drivers. The goal is to find the shortest route between multiple locations to minimize fuel consumption and time. We can apply a shortest-path algorithm (like Dijkstra’s algorithm) to solve this problem.

Question: How can algorithms like Dijkstra’s be used to optimize the delivery route? Answer:

By modeling the delivery locations as nodes and the roads as edges, Dijkstra’s algorithm can calculate the shortest path from the warehouse (starting node) to each delivery location (end node). This minimizes fuel consumption and time, leading to more efficient operations.

Flowcharts and Pseudocode

Flowcharts and Pseudocode

Flowcharts: A flowchart is a diagram that represents the sequence of steps in an algorithm. It uses standard symbols like rectangles (process), diamonds (decisions), and arrows (flow of control) to illustrate the flow.

Pseudocode: Pseudocode is a high-level description of an algorithm using plain language and structured programming constructs. It helps bridge the gap between an algorithm and actual code by describing the logic without using strict syntax.

Example: Business Profit Calculation Algorithm

We want to calculate the total profit for a business based on revenue and costs.

Pseudocode:


// Inputs: revenue, cost
START
    READ revenue
    READ cost
    profit = revenue - cost
    IF profit > 0 THEN
        PRINT "Profit is: " + profit
    ELSE
        PRINT "No profit."
END

Variables, Constants and Datatypes

Variables, Constants, and Datatypes

  • Variables: A variable is a storage location identified by a name that can hold data that may change during the execution of the program.
  • Constants: A constant is a value that cannot be changed once defined.
  • Data Types: Data types define the kind of data a variable can hold, such as integers, floating-point numbers, characters, or booleans.
Business Case Scenario - Inventory Management:

A business needs to track product quantities in a warehouse. It uses variables to store the number of items, and constants to store the maximum storage capacity of the warehouse.

Question: How would you define the inventory variables and constants in a Java program? Answer: 

public class InventoryManagement {
    // Constant for maximum storage capacity
    final int MAX_CAPACITY = 1000;
    
    // Variable to store the current stock
    int currentStock;

    public InventoryManagement(int stock) {
        this.currentStock = stock;
    }

    public void addStock(int quantity) {
        if (currentStock + quantity <= MAX_CAPACITY) {
            currentStock += quantity;
            System.out.println("Stock updated. Current stock: " + currentStock);
        } else {
            System.out.println("Cannot add stock. Exceeds max capacity.");
        }
    }
}
Operators and Expressions

Operators and Expressions

  • Arithmetic Operators: Used to perform mathematical calculations (+, -, *, /, %).
  • Relational Operators: Used to compare values (==, !=, >, <, >=, <=).
  • Logical Operators: Used for logical operations (&&, ||, !).
Business Case Scenario - Employee Salary Calculation:

We need to calculate the total salary of an employee by combining basic salary, bonuses, and deducting taxes.

Question: How can arithmetic and relational operators be used to calculate the salary? Answer: 

public class SalaryCalculator {
    public static void main(String[] args) {
        double basicSalary = 50000;
        double bonus = 5000;
        double tax = 0.2;  // 20% tax rate

        // Calculate gross salary
        double grossSalary = basicSalary + bonus;
        
        // Calculate net salary after tax
        double netSalary = grossSalary - (grossSalary * tax);
        
        System.out.println("Net Salary: " + netSalary);
    }
}
